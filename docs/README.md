# What is Gr8brik ?
Gr8brik is an all-in browser and online Lego modeler built with LdrawLoader and a bunch of other tools.
# Scripts and Plugins used
- JQuery
- ThreeJS
- LdrawLoader
- OrbitControls
- TransformControls
- SpectrumJS
